## 🎯 הבעיה → הפתרון

ללמוד קוד מורס או להבין קידוד שורות קוד באוזן — קשה בלי כלי חזותי ושמיעתי טוב. **SignalLab** מאחד שלושה כלים לפלטפורמה אחת: בונה מילים בקוד מורס עם ויזואליזציה והשמעה, נגן שמשמיע שורות קוד כמו מורס, ומסמך חקירה אינטראקטיבי של אירוע מורס אמיתי (7500).

## ✨ מה בפרויקט

| כלי | קובץ | תיאור |
|-----|------|--------|
| 📡 MorseViz | `morse.html` | בונה מילים בקוד מורס — ויזואליזציה + השמעה |
| ▶️ CodePlayer | `code.html` | משמיע שורות קוד עם צלילים — כמו MorseViz לקוד |
| 📋 מסמך חקירה | `brief.html` | מורס, רעש לבן וחקירת אירוע 7500 |
