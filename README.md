# SignalLab — מעבדת אותות

מערכת מאוחדת שמאחדת **MorseViz**, **CodePlayer** ומסמך חקירת מורס לפלטפורמה אחת.

## עברית

| כלי | קובץ | תיאור |
|-----|------|--------|
| 📡 MorseViz | `morse.html` | בונה מילים בקוד מורס — ויזואליזציה + השמעה |
| ▶️ CodePlayer | `code.html` | משמיע שורות קוד עם צלילים — כמו MorseViz לקוד |
| 📋 מסמך | `brief.html` | מורס, רעש לבן וחקירת אירוע 7500 |

**הרצה:** פתח `index.html` בדפדפן — ללא שרת.

גרסאות ישנות מכפילויות בשולחן העבודה נשמרו ב-`archive/`.

## English

Unified platform combining Morse code visualization, code line audio player, and an aviation/morse investigation brief. Single hub at `index.html`, Hebrew RTL UI, Web Audio API.

**Repos merged:** formerly separate `morseviz` + `code-player` — now one system.
